Tablas - Sencillas

Para todas las tablas utilizadas en Pipoka se debe implementar las clases css .table .table-pipoka, ademas de estar dentro de un div contenedor que lleva por clase .content-tables . Estas clases css le dan un estilio limpio y recomendado a las tablas de la plantilla. Todas las caracteristticas y estilos de Bootstarp pueden ser aplicadas ya que Pipoka se basa en Bootstrap.

			
	<div class="content-tables">
		<table clas="table table-pipoka">

		</table>
	</div>
