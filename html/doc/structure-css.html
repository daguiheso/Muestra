Tabla - Sencilla

Para todas las tablas utilizadas en Pipoka se debe implementar las clases css .table .table-pipoka, ademas de estar dentro de un div contenedor que lleva por clase .content-tables . Estas clases css le dan un estilio limpio y recomendado a las tablas de la plantilla. Todas las caracteristticas y estilos de Bootstarp pueden ser aplicadas ya que Pipoka se basa en Bootstrap.

			
	<div class="content-tables">
		<table clas="table table-pipoka">

		</table>
	</div>

Ademas de las clases perzonalizadas de Pipoka es posible y recomendable utilizar las clases CSS ofrecidas por Bootstrap para las tablas.

                
    <table class="table table-pipoka table-bordered table-striped table-condensed table-hover">

Subtitulos para Cabeceras de Tablas con filas y columnas combinadas

Los subtitulos para tablas con este tipo de cabecera se les aplica la clase css .subtitle-table-pipoka. Este estilo le da una suavidad al peso de la fuente con respecto a los titulos.

Und. PELLENTENQUE HABITAN SECTOR
Promedio
Ult. 5 años		 2013 Promedio
Ult. 5 años		 2013 
            
    <div class="content-tables">
        <div class="table-responsive">
            <table class="table table-pipoka table-bordered table striped table-condensed table-hover">
                <thead>
                    <tr>
                        <th rowspan="2"></th>
                        <th rowspan="2">Und.</th>
                        <th colspan="2">PELLENTENQUE HABITAN</th>
                        <th colspan="2">SECTOR</th>
                        <tr> 
                            <th class="subtitle-table-pipoka">Promedio <br> Ult. 5 años</th>
                            <th class="subtitle-table-pipoka">2013</th>
                            <th class="subtitle-table-pipoka">Promedio <br> Ult. 5 años</th>
                            <th class="subtitle-table-pipoka">2013</th>
                        </tr>                        
                    </tr>
                </thead>            
            </table>
        </div>
    </div>

Nota: En las tablas de tipo DataTable no es recomendable aplicar clases CSS de Bootstrap, ya que se presentaria una sobreescritura de clases entre librerias. DataTable ofrece sus propias clases y estilos CSS.
