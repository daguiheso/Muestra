Estructura General

La plantilla PIPOKA se ha construido en base a Bootstrap. Así que todo lo que encuentre en Bootstrap, se encuentra en PIPOKA.

PIPOKA requiere una hoja de estilos CSS y las importaciones de Javascript. Copie el código de abajo para empezar en su proyecto.

			
    <!DOCTYPE html>
    <html lang="en">
    <head>
        <meta charset="UTF-8">
        <title>MENU</title>
        <meta name="viewport" content="width=device-width, minimum-scale=1,maximum-scale=1">
        <link rel="stylesheet" type="text/css" href="assets/css/style.css">
    </head>
    <body>
        <script src="assets/js/all.js"></script>
    </body>
    </html>

Body

El body se compone de un <nav></nav> que contiene el header global que se utilizara en toda vista o template, otro <nav></nav> que contiene el menu, un <div id="content"> </div> que sera el contenedor en todas las vistas o templates y por ultimo un <footer> </footer>.

La estructura de la plantilla general es la misma en toda la plantilla. Esta es la estructura general

			
    <nav>
       ~ Contenido del header va Aqui ~
    </nav>
    <nav>
       ~ Contenido del menu  va Aqui ~
    </nav>
    <div id="content">   
       ~ Contenido de la vista va Aqui ~
    </div>

Compilación

Lo primero es instalar globalmente el paquete http-server para crear un servidor local.

            
    npm install -g http-server

Una vez instalado, se debe abrir el proyecto (plantilla Pipoka) desde la consola y ejecutar el comando:

            
    http-server

Este comando mantendra un servidor en la maquina local, para efectos de desarrollo.

Pipoka plantilla utiliza Gulp.js con Laravel-elixir para fines de compilacion de Less y JavaScript.

Gulp.js es un build system(sistema de construcción) que permite automatizar tareas comunes de desarrollo, tales como la minificación de código JavaScript, recarga del navegador, compresión de imágenes, validación de sintaxis de código y un sin fin de tareas más.

Laravel Elixir hace aun mas facil el trabajo de Gulp, soportando varios archivos CSS comunes y pre-procesadores de JavaScript.

Mas funcionalidades de Gulp.js aqui (Tutorial).

Pasos para compilar

  • Acceder a folder de la plantilla pipoka desde la consola
  • Ejecutar comando: gulp 
            
    > gulp  ==> Compila
    > gulp watch ==> Compila y se mantiene escuchando cambios
