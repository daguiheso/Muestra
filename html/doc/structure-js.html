DataTables

DataTables es una biblioteca externa que añade funcionalidades de interacción avanzada a cualquier tabla HTML. DataTables ofrece una documentación y ejemplos en su página web.

Empezando

Dependencias

Lo primero es importar los archivos JavaScript y CSS correspondientes para DataTables en nuestro gulpfile.js y compilar.

Es importante el orden de las dependencias, siempre se debe mantener el orden mostrado a continuación (1. jQuery 2. bootstrap.js ...)

			
	var elixir = require('laravel-elixir');


	elixir(function(mix) {
	    mix.less("style.less", "assets/css");

	    mix.scripts([
	    	"libs/jquery/jquery.js",
	    	"libs/bootstrap/bootstrap.js",
	    	"libs/datatable/jquery.dataTables.js",
	    ], "assets/js", "resources/assets/js");
	});

Extensiones para DataTables

Existen una serie de extensiones para DataTables, en esta plantilla utlilizamos solo las siguientes.

  • TableTools

    Esta extension provee las opciones de guardar los DataTables en formato CSV, PDF y Excel

  • FixedColumns

    Esta extension permite el posicionamiento fijo de las columnas de un DataTable.
