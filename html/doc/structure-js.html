Tabla Sencilla con JavaScript

Tablas basadas en bootstrap que por medio de JavaScript permiten ampliar o reducir una tabla, esto a través de la aparición y ocultamiento de filas.


En este ejemplo se le asigna la clase css .showHide a los elementos del DOM <tr> que representan las filas que alternaran entre display:block; y display:none;Por defecto todos los elementos del DOM tienen display:block;. Por ende si se quiere cambiar este comportamiento se debe asignar la propiedad css display:none;al elemento deseado.
Siguiendo con el ejemplo, a una fila de la tabla se le asigna la clase css .is-Inactive que aplica la propiedad display:none; Lo cual provoca que por defecto esta fila este oculta y al detectar el evento .click se muestre.

        // DECLARANDO VARIABLES HACIENDO REFERENCIAS A ELEMENTOS DEL DOM

        var $btnCollapsedExample = $("#btn-collapsed-Example"),  // ==> BOTON DEL DOM CON ID btn-collapsed-Example
            $showHideBg1Tab1 = $(".showHide");  // ELEMENTOS DEL DOM CON LA CLASE .showHide


        // FUNCION MOSTRAR OCULTAR ELEMENTOS

        function showHide() {
            $showHideBg1Tab1.toggle();  // METODO .toggle() PARA ALTERNAR ENTRE show Y hide.
            return false; // EVITA EL COMPORTAMIENTO POR DEFECTO DEL EVENTO
        } 

        // EVENTO CLICK

        $btnCollapsedExample.click(showHide);  // AL DETECTAR EVENTO CLICK EJECUTA FUNCION showhide

Tabla con DataTables

DataTables es una biblioteca externa que añade funcionalidades de interacción avanzada a cualquier tabla HTML. DataTables ofrece una documentación y ejemplos en su página web.

Empezando

Dependencias

Lo primero es importar los archivos JavaScript y CSS correspondientes para DataTables en nuestro gulpfile.js y compilar.

Es importante el orden de las dependencias, siempre se debe mantener el orden mostrado a continuación (1. jQuery 2. bootstrap.js ...)

			
	var elixir = require('laravel-elixir');


	elixir(function(mix) {
	    mix.less("style.less", "assets/css");

	    mix.scripts([
	    	"libs/jquery/jquery.js",
	    	"libs/bootstrap/bootstrap.js",
	    	"libs/datatable/jquery.dataTables.js",
	    ], "assets/js", "resources/assets/js");
	});

Extensiones para DataTables

Existen una serie de extensiones para DataTables, en esta plantilla utlilizamos solo las siguientes.

  • TableTools

    Esta extension provee las opciones de guardar los DataTables en formato CSV, PDF y Excel

  • FixedColumns

    Esta extension permite el posicionamiento fijo de las columnas de un DataTable.

JavaScript

Una vez compilado con Gulp.js con las dependencias, es necesario inicializar las tablascon DataTable.

D3 es una libreria JavaScript para la manipulación de documentos basados en datos. D3 facilita el render de datos usando HTML, SVG y CSS. Combina componentes de visualización de gran alcance y un enfoque basado en datos de la manipulación del DOM.
