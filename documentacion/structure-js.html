DataTables

DataTables es una biblioteca externa que añade funcionalidades de interacción avanzadas a cualquier tabla HTML. DataTables ofrece una documentación y ejemplos en su página web

Empezando

Dependencias

Lo primero es importar los archivos JavaScript y CSS en nuestro gulpfile.js y compilar.

			
	var elixir = require('laravel-elixir');


	elixir(function(mix) {
	    mix.less("style.less", "assets/css");

	    mix.scripts([
	    	"libs/jquery/jquery.js",
	    	"libs/bootstrap/bootstrap.js",
	    	"libs/datatable/jquery.dataTables.js",
	    ], "assets/js", "resources/assets/js");

	});
