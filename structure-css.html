Tipografia

Ademas de los estilos de Bootstrap, Pipoka tiene opciones adicionales.

  • Esto es un H1
  • Esto es un H2
  • Esto es un H3
  • Esto es un H4
  • Esto es un H5
  • Esto es un H6
			
	<h1 class="title-h1">Esto es un H1</h1>
	<h2 class="title-h2">Esto es un H2</h2>
	<h3 class="title-h3">Esto es un H3</h3>
	<h4 class="title-h4">Esto es un H4</h4>
	<h5 class="title-h5">Esto es un H5</h5>
	<h6 class="title-h6">Esto es un H6</h6>

Tablas

Para todas las tablas utilizadas en Pipoka se debe implementar la clase .table junto a la clase .table-pipoka, ademas de estar dentro de un div contenedor que lleva por clase .content-tables. Estas clases css le dan un estilio limpio y recomendado a las tablas de la plantilla. Todas las caracteristticas y estilos de Bootstarp pueden ser aplicadas ya que Pipoka se basa en Bootstrap.

			
	<div class="content-tables">
		<table clas="table table-pipoka">

		</table>
	</div>
