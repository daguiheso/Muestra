Introducción

Estructura General

La plantilla PIPOCA se ha construido en base a Bootstrap. Así que todo lo que encuentre en Bootstrap, se encuentra en PIPOCA.

HTML

PIPOCA requiere una hoja de estilos CSS y las importaciones de Javascript. Copie el código de abajo para empezar en su proyecto. 

			
				<!DOCTYPE html>
				<html lang="en">
				<head>
					<meta charset="UTF-8">
					<title>Introducción PIPOCA</title>
					<link rel="stylesheet" type="text/css" href="assets/css/style.css">
				</head>
				<body>
					<header class="header">
					  <h1 class="name">Latinvestco</h1>
					  <ul class="nav nav-pills nav-stacked">
					    <li>
					      <a href="/introduccion">Comenzando</a>
					      <a href="">Estructura CSS</a>
					      <a href="">Componentes</a>
					      <a href="">Javascript</a>
					    </li>
					  </ul> 
					</header>
				  <script src="assets/js/all.js"></script>
				</body>
				</html>
